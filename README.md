# Assignment 1 - Python Pandas Shopping Analysis

## Overview

This project is submitted as **Assignment 1** for the Celebal Technologies Internship.

The objective of this assignment is to perform data cleaning and analysis on a shopping dataset using the Pandas library in Python.

## Files Included

- `Python_Pandas_Shopping_Analysis.ipynb` – Jupyter Notebook containing the complete analysis.
- `shopping_raw.csv` – Original dataset.
- `shopping_cleaned.csv` – Cleaned dataset generated after preprocessing.

## Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

## Tasks Performed

- Loaded the shopping dataset
- Explored the dataset using Pandas
- Checked data types and summary statistics
- Handled missing values
- Removed duplicate records
- Cleaned and preprocessed the data
- Exported the cleaned dataset

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/Sakshixa07/CelebalAssignments.git
```

2. Navigate to the Assignment1 folder:

```bash
cd CelebalAssignments/Assignment1
```

3. Install the required libraries:

```bash
pip install pandas numpy
```

4. Open the notebook:

```bash
jupyter notebook Python_Pandas_Shopping_Analysis.ipynb
```

5. Run all cells to generate the cleaned dataset.
